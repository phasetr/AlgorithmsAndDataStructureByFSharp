## 二分探索
n 個の整数を含む数列 S と、
q 個の異なる整数を含む数列 T を読み込み、
T に含まれる整数の中で S に含まれるものの個数 C を出力するプログラムを作成してください。
## 入力
１行目に n、２行目に S を表す n 個の整数、３行目に q、４行目に T を表す q 個の整数が与えられます。
## 出力
C を１行に出力してください。
## 制約
- S の要素は昇順に整列されている
- n≤100,000
- q≤50,000
- 0≤Sの要素≤109
- 0≤Tの要素≤109
- T の要素は互いに異なる
## 入力例 1
```
5
1 2 3 4 5
3
3 4 1
```
## 出力例 1
```
3
```
## 入力例 2
```
3
1 2 3
1
5
```
## 出力例 2
```
0
```
## 入力例 3
```
5
1 1 2 2 3
2
1 2
```
## 出力例 3
```
2
```
