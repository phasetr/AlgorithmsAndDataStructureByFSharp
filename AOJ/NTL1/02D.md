## Division of Big Integers
２つの整数AとBの商ABを求めてください。
ただし、小数点以下は切り捨てるものとします。
## 入力
２つの整数A, Bが空白区切りで１行に与えられる。
## 出力
AとBの商を１行に出力せよ。
## 制約
- −1×101000≤A,B≤101000
- B≠0
## 入力例1
```
5 8
```
## 出力例1
```
0
```
## 入力例2
```
100 25
```
## 出力例2
```
4
```
## 入力例3
```
-1 3
```
## 出力例3
```
0
```
## 入力例4
```
12 -3
```
## 出力例4
```
-4
```
