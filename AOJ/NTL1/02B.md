## Difference of Big Integers
２つの整数AとBの差A−Bを求めてください。
## 入力
２つの整数A, Bが空白区切りで１行に与えられる。
## 出力
AとBの差を１行に出力せよ。
## 制約
- −1×10^{100000}≤A,B≤10^{100000}
## 入力例1
```
5 8
```
## 出力例1
```
-3
```
## 入力例2
```
100 25
```
## 出力例2
```
75
```
## 入力例3
```
-1 -1
```
## 出力例3
```
0
```
## 入力例4
```
12 -3
```
## 出力例4
```
15
```
