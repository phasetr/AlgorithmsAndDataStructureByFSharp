## D - Binomial Coefficients / 問題文
n 個のものから順番を無視して r 個を選ぶ場合の数を comb(n,r) と書くことにします。
n 個の非負の整数 a1​,a2​,...,an​ から 2 つの数 ai​>aj​ を comb(ai​,aj​) が最大になるように選んで下さい。
最大になる組が複数ある場合、どれを選んでも構いません。
## 制約
- 2≤n≤10^5
- 0≤ai​≤10^9
- a1​,a2​,...,an​ は互いに相異なる
- 入力はすべて整数
## 入力
入力は以下の形式で標準入力から与えられる。

```
n
a1 a2 ... an
```
## 出力
選んだ 2 つの数を空白区切りで降順に出力せよ。
## 入力例 1
```
5
6 9 4 2 11
```
## 出力例 1
```
11 6
```

それぞれ計算すると

- comb(4,2)=6
- comb(6,2)=15
- comb(6,4)=15
- comb(9,2)=36
- comb(9,4)=126
- comb(9,6)=84
- comb(11,2)=55
- comb(11,4)=330
- comb(11,6)=462
- comb(11,9)=55

となるため、
11 と 6 を出力します。
## 入力例 2
```
2
100 0
```
## 出力例 2
```
100 0
```
