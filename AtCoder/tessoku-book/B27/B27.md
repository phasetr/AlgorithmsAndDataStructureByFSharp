## B27 - Calculate LCM
整数 A と B の最小公倍数（ A,B 両方の倍数であるような正の整数 x のうち最小のもの）を出力してください。
## 制約
* A,B は整数
* 1 \le A,B \le 10^9
## 入力
入力は以下の形式で標準入力から与えられる。

```
A B
```
## 出力
答えを整数として出力せよ。
## 入力例 1
```
25 30
```
## 出力例 1
```
150
```
## 入力例 2
```
998244353 998244853
```
## 出力例 2
```
996492287418565109
```

答えは 32bit 符号付き整数に収まらない可能性があります。
