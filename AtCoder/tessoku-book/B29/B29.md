## B29 - Power Hard
a^b を 1000000007 で割った余りを出力せよ。
## 制約
* a,b は整数
* 1 \le a \le 10^9
* 1 \le b \le 10^{18}
## 入力
入力は以下の形式で標準入力から与えられる。

```
a b
```
## 出力
答えを整数として出力せよ。
## 入力例 1
```
6 3
```
## 出力例 1
```
216
```
## 入力例 2
```
123456789 123456789012345678
```
## 出力例 2
```
3599437
```

1000000007 で割った余りを出力してください。
