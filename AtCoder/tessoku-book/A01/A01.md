## A01 - The First Problem
整数 N が与えられるので、
一辺の長さが N であるような正方形の面積を出力するプログラムを作成してください。
## 制約
* N は 1 以上 100 以下の整数
## 入力
入力は以下の形式で標準入力から与えられます。

```
N
```
## 出力
正方形の面積を整数で出力してください。
## 入力例 1
```
2
```
## 出力例 1
```
4
```

一辺の長さが 2 である正方形の面積は 2 \times 2 = 4 です。
## 入力例 2
```
8
```
## 出力例 2
```
64
```

一辺の長さが 8 である正方形の面積は 8 \times 8 = 64 です。
## 入力例 3
```
100
```
## 出力例 3
```
10000
```

一辺の長さが 100 である正方形の面積は 100 \times 100 = 10000 です。
