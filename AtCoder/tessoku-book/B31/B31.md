## B31 - Divisors Hard
1 以上 N 以下の整数のうち、 3,5,7 のいずれかで割り切れるものは何個ありますか。
## 制約
* N は 1 以上 10^{12} 以下の整数
## 入力
入力は以下の形式で標準入力から与えられる。

```
N
```
## 出力
答えを整数として出力してください。
## 入力例 1
```
10
```
## 出力例 1
```
6
```

10 以下の整数のうち 3,5,7 のいずれかで割り切れるものは、 3,5,6,7,9,10 の 6 個です。
## 入力例 2
```
210
```
## 出力例 2
```
114
```
## 入力例 3
```
100000000000
```
## 出力例 3
```
54285714286
```
