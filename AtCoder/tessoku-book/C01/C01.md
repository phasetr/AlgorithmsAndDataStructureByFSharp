## C01 - Tax Rate
ある商品の税抜価格は N 円です。
消費税率が 10 \% であるとき、
この商品の税込価格が何円であるかを出力するプログラムを作成してください。
ただし、N は 100 の倍数であるとします。
## 制約
* 1 \leqq N \leqq 10^6
* N は 100 の倍数
## 入力
入力は以下の形式で標準入力から与えられます。

```
N
```
## 出力
商品の税込価格を標準出力に 1 行で出力してください。
## 入力例 1
```
800
```
## 出力例 1
```
880
```
