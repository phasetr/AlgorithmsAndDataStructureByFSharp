## C05 - Lucky Numbers
4, 7 のみからなる 10 桁の整数をラッキー数と呼ぶことにします。小さい方から数えて N 番目のラッキー数はいくつですか。
## 制約
* N は 1 以上 1024 以下の整数
## 入力
入力は以下の形式で標準入力から与えられます。

```
N
```
## 出力
小さいほうから数えて N 番目のラッキー数を標準出力に 1 行で出力してください。
## 入力例 1
```
5
```
## 出力例 1
```
4444444744
```
