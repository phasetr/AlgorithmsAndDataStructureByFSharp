## C09 - Taro's Vacation 
太郎君の夏休みは N 日間あり、
i 日目に勉強すると A_i だけ実力が上がることが知られています。

しかし、彼は 2 日連続で勉強したくありません。
太郎君が夏休みの間に実力をどれだけ上げられるか、
その最大値を求めるプログラムを作成してください。
## 制約
* 2 \leq N \leq 500000
* 0 \leq A_i \leq 10^9
* 入力はすべて整数
## 入力
入力は以下の形式で標準入力から与えられます。

```
N
A_1 A_2 \cdots A_N
```
## 出力
答えを出力してください。
## 入力例 1
```
5
2 5 3 3 1
```
## 出力例 1
```
8
```

2 日目、4 日目に勉強すると、実力が 5+3=8 上がります。
