## A29 - Power
a^b を 1000000007 (=10^9+7) で割った余りを計算してください。
## 制約
* 1 \le a \le 100
* 1 \le b \le 10^9
* 入力はすべて整数
## 入力
入力は以下の形式で標準入力から与えられます。

```
a b
```
## 出力
答えを整数として出力してください。
## 入力例 1
```
5 23
```
## 出力例 1
```
871631629
```

5^{23}=11920928955078125 ですが、
これを 10^9+7 で割った余りである 871631629 を出力してください。
## 入力例 2
```
97 998244353
```
## 出力例 2
```
934801994
```

b の値は大きくなることがあります。
