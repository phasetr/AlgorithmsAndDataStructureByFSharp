## B01 - A+B Problem
整数 A と B が与えられます。
A+B の値を出力してください。
## 制約
* A は 1 以上 100 以下の整数
* B は 1 以上 100 以下の整数
## 入力
入力は以下の形式で標準入力から与えられます。

```
A B
```
## 出力
答えを整数で出力してください。
## 入力例 1
```
1 2
```
## 出力例 1
```
3
```

1+2=3 です。
## 入力例 2
```
77 23
```
## 出力例 2
```
100
```

77+23=100 です。
## 入力例 3
```
100 100
```
## 出力例 3
```
200
```

100+100=200 です。
